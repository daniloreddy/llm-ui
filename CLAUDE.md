# CLAUDE.md

This file provides guidance to Claude Code when working with code in this repository.

## Project Overview

FastAPI backend (Python) acting as a CORS proxy + pure JS SPA frontend.  
The backend serves static files and proxies LLM requests. No JS build step.

## Running the App

```bash
bash scripts/run.sh   # Linux/macOS
scripts\run.bat       # Windows
```

Open `http://127.0.0.1:8050` in the browser.

The run scripts:
1. Download the Tailwind CSS binary to `bin/` if missing (`setup-tailwind.sh/.bat`)
2. Generate `static/tw.css` from `static/input.css` (`--minify`)
3. Start uvicorn

## Architecture

### Backend (`app/`)

| File | Role |
|------|------|
| `main.py` | FastAPI app — auth middleware + routes (`/login`, `/auth/login`, `/auth/logout`), `/api/chat` (SSE proxy), `/api/config` (GET/PUT), static file serving |
| `auth.py` | `AuthManager`: scrypt password hashing, JWT session cookies, per-IP + global rate limiting, trusted-proxy IP resolution |
| `config.py` | `ConfigManager`: `asyncio.Lock`, atomic write via `.tmp` + `Path.replace()`, `copy.deepcopy` on read/write |
| `proxy.py` | `proxy_stream()`: httpx async streaming proxy; `_clamp()`, `_build_payload()` |

Config persisted to `data/config.json`. Auth state in `data/auth.json`. Docs disabled (`openapi_url=None`).

### Environment (`.env`)

Loaded via `python-dotenv` at startup. Copy `.env.example` → `.env` and edit as needed.

| Variable | Default | Description |
|---|---|---|
| `TRUSTED_PROXIES` | `127.0.0.1` | Comma-separated IPs of trusted reverse proxies (Cloudflare Tunnel, Apache, nginx) |
| `AUTH_SECURE_COOKIE` | _(unset)_ | Set to `1` to force the session cookie `Secure` flag (not needed behind Cloudflare Tunnel) |

### First run

```bash
python scripts/set_password.py
```

Must be run once before the app accepts any login. Password hash stored in `data/auth.json`.

### Frontend (`static/`)

| File | Role |
|------|------|
| `index.html` | SPA shell — Chat, Config, Guida tabs; `data-i18n` attributes for translations |
| `app.js` | All frontend logic — state, i18n, streaming, multi-panel, attachments |
| `style.css` | Custom styles (bubbles, console drawer, think blocks, copy-code btn, etc.) |
| `input.css` | Tailwind v4 entry point (`@import "tailwindcss"` + `@source` directives) |
| `tw.css` | Generated by Tailwind CLI — gitignored |
| `i18n/it.json` / `i18n/en.json` | Translation strings (56 keys each) |
| `guide/it.html` / `guide/en.html` | Lazy-loaded guide HTML fragments |

### State model (`app.js`)

Persisted to `data/config.json` via `PUT /api/config`:

```
endpoints[]    — array of endpoint objects
layout         — { count: 1-4, slots: [id|null, ...], broadcastMode: bool }
prefs          — { fontFamily, googleFont, lang }
```

Per-panel runtime (not persisted): `history[]`, `attachedFiles[]`, `logs[]`, `isStreaming`, `abortController`, `historyCursor`, `promptDraft`, `lastUserMsg`, `lastAssistantWrapper`.

### Endpoint schema

```
id, name, serverUrl, useRawUrl, model, apiKey,
systemPrompt, temperature, top_p, max_tokens, repeat_penalty
```

### Message format

- Text/PDF: appended as `[File allegato: name]\ncontent` to the user text string
- Images: OpenAI multimodal array `[{type:"text",...},{type:"image_url",image_url:{url:"data:..."}}]`
- `Authorization: Bearer <apiKey>` added only when `apiKey` is non-empty

### Streaming

`/api/chat` → httpx async streaming → SSE to browser.  
Frontend: `fetch` + `ReadableStream`, split on `\n`, parse `data: <json>`, append delta. `[DONE]` terminates.

### i18n

- `t(key, vars?)` — looks up `_t` (loaded JSON dict), falls back to key
- `loadTranslations(lang)` — `fetch('i18n/${lang}.json')`
- `applyTranslations()` — walks `[data-i18n]`, `[data-i18n-placeholder]`, `[data-i18n-title]`
- Adding a language: create `i18n/xx.json` + `guide/xx.html` + one `<option>` in HTML — no code changes

### Tailwind CSS

Binary downloaded to `bin/` (gitignored) by `scripts/setup-tailwind.sh` / `.bat`.  
Command: `./bin/tailwindcss -i static/input.css -o static/tw.css --minify`  
`tw.css` is gitignored — regenerated on every `check` or `run`.

### Docker / CI

| File | Role |
|------|------|
| `Dockerfile` | Builds image: installs deps, copies `app/`, `static/`, `scripts/`, generates `tw.css` at build time via `scripts/gen-tw.py` |
| `docker-compose.yml` | Server deploy — pulls `ghcr.io/daniloreddy/llm-ui:latest`, mounts `./data`, loads `.env` |
| `.github/workflows/docker.yml` | Builds and pushes multi-arch image (amd64 + arm64) to GHCR on push to `main` or `v*` tags |

First-run password setup in Docker:
```bash
docker compose exec llm-ui python scripts/set_password.py
```
